To run the application, run university _search.py using Python 3.
Required packages:
requests
json
random
flask - Flask

This application will help you look up a list of universities based on the country, upper and lower ranking limit, and then provide a search results list with an option whether to see the address or not.

To start the search, you will need to provide a 2-character country code. You may look up the code in need at this webpage https://countrycode.org. Use the 2-character ISO code from the ISO code column. Common 2-character ISO code:
USA - US
China - CH
Japan - JP
Germany - DE
India - IN
United Kingdom - GB
France - FR
Brazil - BR
Italy - IT
Canada - CA
Russia - RU

Follow the prompt. The final results will be shown in a browser using Flask.
